Bagaimana?

1. Buat struktur Html susunan kartu jumlah akar(bilangan kuadrat genap)

2. Buat papan skor berisi: gerak, waktu, rating

3. Buat tutorial main agar pemain tidak bingung

4. Styling, buat file css agar tampilan html tersusun sesuai keinginan

5. Buat file javascript, codingan merupakan mekanik game tersebut

6. Buat css pengaturan agar responsive di semua perangkat, buat secara pribadi di masing masing html agar tidak saling bertabrakan atau crash (ini yang cukup mumeti)

7. Tambahkan animasi GSAP agar lebih canggih, dan tambahkan audio effect

8. Tambahkan Loader (halaman loading), agar loading menjadi lebih menarik

9. Siapkan aset-aset pendukung

10. Buat E-Certi untuk unlock minigame selanjutnya

11. Pusing gw ngodingnya, kurang lebih semingguan baru kelar

12. Awas aja kalo dibilang jelek/gajelas, namanya aja mini game, berjam jam ngodingya sampai lembur gw

13. Ditambah lagi harus ngecilin file size satu per satu yang jumlahnya banyak banget agar tidak ngelag
